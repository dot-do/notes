## Performance Analysis

### Expected Performance (p95)

| Operation | Target | Notes |
|-----------|--------|-------|
| Token Validation | <5ms | Pure crypto, very fast |
| API Key Validation | <10ms | bcrypt overhead (intentional) |
| Session Lookup | <5ms | Single DB RPC call |
| User Lookup | <5ms | Single DB RPC call |
| GitHub OAuth | <200ms | External API dependency |

### Bottlenecks

**1. API Key Validation (~10ms)**
- **Cause:** bcrypt work factor 10
- **Intentional:** Security vs. performance tradeoff
- **Mitigation:** Cache validated keys for 5-10 minutes

**2. Multiple DB RPC Calls**
- **Scenario:** Lookup user â†’ lookup session â†’ lookup API key
- **Mitigation:** Batch operations when possible

### Optimization Strategies

**1. Token Caching:**
```typescript
// Cache validated tokens for 5 minutes
const tokenCache = new Map<string, { user: User; expires: number }>()

async function validateTokenCached(token: string) {
  const cached = tokenCache.get(token)
  if (cached && cached.expires > Date.now()) {
    return cached.user
  }

  const user = await env.AUTH.validateToken(token)
  if (user) {
    tokenCache.set(token, { user, expires: Date.now() + 300_000 })
  }
  return user
}
```

## Security Analysis

**1. JWT Attacks**

- âŒ **Replay Attacks:** Expiration checking (7 days)
- âŒ **Algorithm Confusion:** Only HS256 accepted
- âŒ **Key Disclosure:** Secret stored in Cloudflare secret

**2. API Key Attacks**
- âŒ **Brute Force:** 256-bit entropy (2^256 possibilities)
- âŒ **Rainbow Tables:** bcrypt salting
- âŒ **Timing Attacks:** Constant-time bcrypt.compare()
- âŒ **Format Injection:** Regex validation

**3. Session Attacks**
- âŒ **Session Hijacking:** HttpOnly cookies (no JS access)
- âŒ **CSRF:** SameSite=Lax cookies
- âŒ **Session Fixation:** UUID v4 (unpredictable)
- âŒ **XSS Cookie Theft:** HttpOnly + Secure flags

    const cached = tokenCache.get(token)
    if (cached && cached.expires > Date.now()) {
      c.set('user', cached.user)
      return await next()
    }

    // Validate token
    const user = await c.env.AUTH.validateToken(token)
    if (!user) {
      return c.json({ error: 'Invalid token' }, 401)
    }

    // Cache for 5 minutes
    tokenCache.set(token, {
      user,
      expires: Date.now() + 300_000
    })

    c.set('user', user)
    await next()
  }
}
