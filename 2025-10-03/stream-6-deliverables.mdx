## Quick Reference

### React-ink API

```typescript
// Render MDX to CLI
import { renderMdxToInk, defaultInkComponents } from 'mdxe/outputs/react-ink'

const result = await renderMdxToInk(mdxContent, {
  components: defaultInkComponents,
  scope: { data },
})
```
