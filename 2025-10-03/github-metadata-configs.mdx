## Social Preview Image Specifications

### Technical Requirements
- **Dimensions:** 1280x640px (2:1 aspect ratio)
- **Format:** PNG or JPG
- **File Size:** Under 1MB (optimize with TinyPNG)
- **Upload Location:** Repository Settings → Social preview → Upload an image

### Design Template

```
┌────────────────────────────────────────────────────────────────┐
│                                                                │
│  .do                                          [Tech Icons]     │
│                                                                │
│                                                                │
│                  [REPOSITORY NAME]                             │
│                                                                │
│              [One-line description]                            │
│                                                                │
│                                                                │
│  Built with [Tech Stack]                                       │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

### Color Palette
- **Primary Blue:** #667eea
- **Secondary Purple:** #764ba2
- **Accent Green:** #10B981
- **Accent Orange:** #F97316
- **Accent Pink:** #EC4899
- **White Text:** #FFFFFF
- **Dark Background:** #1F2937

### Creation Methods

**Option 1: Browser-Based (Fastest)**
1. Create HTML file with template
2. Open in browser at 1280x640 viewport
3. Use browser screenshot tool (full page)
4. Save and optimize

**Option 2: Figma (Most Professional)**
1. Create 1280x640 frame
2. Apply gradient background
3. Add .do logo (top-left)
4. Add repository name (large, center)
5. Add description text (below name)
6. Add tech icons (bottom or right)
7. Export as PNG

**Option 3: Canva (Easiest)**
1. Create custom size: 1280x640px
2. Choose gradient background
3. Add text elements
4. Add tech icons from Elements
5. Download as PNG

**Option 4: Code-Generated (Most Scalable)**
```bash
# Use Playwright or Puppeteer to screenshot HTML template
node scripts/generate-social-images.js --repo=api
```

## Common Issues and Solutions

### Issue: About Section Too Long
**Problem:** GitHub truncates at 120 characters
**Solution:** Prioritize most important keywords first, remove filler words

### Issue: Topics Not Appearing
**Problem:** Topics may take a few minutes to display
**Solution:** Refresh page, wait 5 minutes, check Repository Settings

### Issue: Social Preview Not Showing
**Problem:** Image not uploaded or wrong size
**Solution:** Verify image is exactly 1280x640px, re-upload if needed

### Issue: Website URL Broken
**Problem:** URL not yet deployed or wrong domain
**Solution:** Use placeholder (github.io) until production domain ready

## Batch Application Script

For automated metadata updates via GitHub API:

```bash
#!/bin/bash
# Script to update GitHub repository metadata (requires GitHub CLI)

REPOS=("api" "db" "workers" "sdk" "ai" "agent" "auth" "app")

for repo in "${REPOS[@]}"; do
  echo "Updating $repo..."

  # Update description (About section)
  gh repo edit dot-do/$repo --description "$(cat metadata/${repo}-description.txt)"

  # Update topics
  gh repo edit dot-do/$repo --add-topic "$(cat metadata/${repo}-topics.txt)"

  # Update homepage URL
  gh repo edit dot-do/$repo --homepage "https://${repo}.do"

  echo "✓ Updated $repo"
done

echo "All repositories updated!"
```

**Note:** Social preview images must still be uploaded manually via GitHub UI.
