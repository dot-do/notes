### How to Use GitHub Lists

**Method 1: Direct Navigation**
1. Visit awesome list repository
2. Ctrl+F search for "DA" or "domain authority"
3. Extract all directories with DA > 50
4. Create submission spreadsheet

**Method 2: GitHub Search**
```
site:github.com "awesome" "directory" "DA" "domain authority"
site:github.com "awesome" "saas" "submit" "backlink"
site:github.com "directory submission" "high DA" "dofollow"
```

**Method 3: Automated Extraction**
```bash
# Clone awesome list
git clone https://github.com/mahseema/awesome-saas-directories.git

# Extract URLs with DA scores
grep -E "(DA|DR).*[0-9]{2}" README.md > high_da_directories.txt

# Parse and create CSV
python parse_directories.py high_da_directories.txt > directories.csv
```
